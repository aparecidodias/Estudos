No livro PHP: a Bíblia, Tim Converse explica o que é PHP.
PHP é uma linguagem para a criação de scripts para a web do lado servidor embutidos em HTML, cujo coódigo-font é aberto, e que é compativel com os mais importantes servidores Web (especialmente Apache). OPHP permite incorpora fragmentos de código em páginas HTML normais - código esse que é interpretado á medida que suas páginas HTML normais - código esse que é interpretado á medida que suas páginas são oferecidas aos usuários. O PHP também serve como uma linguagem que "cola", facilitando a conexão de suas paginas Web com o banco de dados do lado servidor.
